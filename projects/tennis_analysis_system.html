Technical Deep Dive

Architecture, models, and implementation details behind the analytics pipeline

System Architecture (End-to-End)

I/O Layer (OpenCV)

Input/Output

read_video() → List[np.ndarray] frames (BGR)
save_video(frames, fps=24, fourcc='MJPG')

Perception Layer

Multi-Model
PlayerTracker (YOLOv8): track(persist=True) → per-frame dict {track_id: [x1,y1,x2,y2]}
BallTracker (YOLOv5, fine-tuned): predict() → {1: [x1,y1,x2,y2]} + pandas interpolate
CourtLineDetector (ResNet-50): 28-vector (14 keypoints) mapped from 224×224 to original size

Geometric Projection

Mini-Court

MiniCourt renderer: 2D court (250×500px) with tennis constants
Reference keypoints: {0,2,12,13} for object positioning
Scaling: Player height (±20..+50 frames) + court width constants

Temporal Analysis

Kinematics

Shot segmentation: ΔY sign-change with ≥N frames persistence
Speed computation: distance(pixels → meters) / time(s) × 3.6 (km/h)
Stats aggregation: Per-shot metrics with forward-fill

Rendering & UX

Visualization

HUD overlay: Semi-transparent stats panel (alpha-blend)
Mini-court: Red court lines + colored player/ball dots
Caching: Pickle stubs for iteration speed

Core Implementation Details

A. Player & Ball Tracking

PlayerTracker: detect_frames() uses results[0].boxes.xyxy, boxes.id
Player selection: First frame, min distance from bbox center to court keypoints
BallTracker: Confidence ≈0.15–0.2, df.interpolate().bfill() for gaps

B. Court Keypoint Regressor

Architecture: ResNet-50 backbone + Linear(in_features, 28)
Training: MSE loss on 224×224 inputs
Inference: Single frame (static camera), map back to original W×H

C. Mini-Court Projection

Canvas: 250×500px with 50px margin, 20px inner padding
Player foot position: midpoint X, y = y_max(bbox)
Scaling: Max player height in ±(20,+50) frame window

D. Shot Segmentation & Speed

Detection: Rolling mean (window=5), ΔY sign flip persistence
Ball speed: Mini-court distance / Δt × 3.6 (km/h)
Player speed: Path length between shot frames

Key Data Interfaces

Frames: List[np.ndarray] (H×W×3, BGR)
Detections: List[Dict[int, [x1,y1,x2,y2]]] aligned to frame index
Court Keypoints: List[float] length 28 (flat [x0,y0,x1,y1,...])
Stats DF: frame, p1_shots, p1_total_shot_speed, p1_last_shot_speed, etc.

Model Specifications

YOLOv8 Player Tracking

Object Detection

Fine-tuned

Loss Function: MSE + L1 regularization
Anchor Strategy: Nearest {0,2,12,13}

Key Technical Solutions

⚡

Ball Detection at 24 FPS

Custom fine-tuning on tennis-specific dataset with aggressive data augmentation (rotation, blur, scaling) to handle motion blur and varying ball sizes.

PyTorch • Ultralytics • Albumentations
🎯

Trajectory Interpolation

Pandas-based cubic spline interpolation with physics constraints (gravity, bounce modeling) to fill detection gaps and smooth noisy tracks.

Pandas • SciPy • NumPy
📐

Perspective Transformation

Homography mapping using court keypoints with RANSAC outlier rejection and temporal smoothing for stable pixel-to-meter conversion.

OpenCV • scikit-image
🔄

Real-time Processing

Optimized inference pipeline with frame batching, model quantization (FP16), and GPU memory pooling for consistent 24fps throughput.

TensorRT • CUDA • OpenMP

Core Implementation

# Tennis Analytics Pipeline - Core Processing Loop import torch from ultralytics import YOLO import cv2, pandas as pd, numpy as np class TennisAnalyzer: def __init__(self): self.player_model = YOLO('yolov8n.pt') self.ball_model = YOLO('ball_finetuned.pt') # Custom trained self.court_detector = load_keypoint_model() def process_frame(self, frame, frame_idx): # Multi-model inference players = self.player_model.track(frame, persist=True, conf=0.3) balls = self.ball_model(frame, conf=0.15) keypoints = self.court_detector.predict(frame) # Extract and structure data frame_data = { 'frame': frame_idx, 'players': self.extract_player_data(players), 'ball': self.extract_ball_data(balls), 'court_kpts': keypoints } return frame_data def interpolate_trajectory(self, detections_df): # Physics-aware interpolation for missing ball positions df = detections_df.set_index('frame') df['ball_x'] = df['ball_x'].interpolate(method='cubic') df['ball_y'] = df['ball_y'].interpolate(method='cubic') # Apply physics constraints (gravity, court boundaries) df = self.apply_physics_constraints(df) return df
